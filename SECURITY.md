# Security Policy

Thank you for helping improve the security of this project.

The **Enterprise MikroTik VPN Guide** is intended to provide production-oriented documentation and reference configurations for secure MikroTik RouterOS VPN deployments.

Security is a core principle of this repository.

# Scope

This repository includes:

- Documentation
- Reference RouterOS configurations
- Deployment examples
- Best practices

It does **not** include:

- Production credentials
- Private keys
- Certificates
- Customer configurations
- Sensitive infrastructure information

---

# Security Principles

The reference configurations included in this repository follow these principles:

- Least Privilege
- Defense in Depth
- Secure Defaults
- Strong Encryption
- Secure Remote Access
- Network Segmentation
- Centralized Authentication
- Logging and Monitoring
- Regular Updates

---

# Responsible Disclosure

Please provide reasonable time to investigate and resolve any reported issue before public disclosure.

Responsible disclosure helps protect users and organizations relying on this documentation.

---

# Best Practices

Before deploying any configuration:

- Review every RouterOS command.
- Adapt IP addressing to your environment.
- Generate unique cryptographic keys.
- Use strong passwords.
- Restrict administrative access.
- Keep RouterOS updated.
- Test configurations before production deployment.
- Backup your RouterOS configuration.

---
